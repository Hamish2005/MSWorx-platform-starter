
# Project Instructions

Build premium, modern landing pages using Next.js, Tailwind CSS, and Framer Motion.

Design rules:
- Use clean spacing and strong visual hierarchy.
- Use an 8px spacing system.
- Avoid generic AI-looking gradients and random colours.
- Use consistent typography sizes.
- Use reusable components where possible.
- Make every page fully responsive.
- Use Framer Motion for scroll reveals, hover animations, and staggered card animations.
- Keep performance high: optimized images, clean components, and no unnecessary dependencies.

When editing:
- Check the project structure first.
- Explain major changes briefly.
- Run the app or build when possible.
